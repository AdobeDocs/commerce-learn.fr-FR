# Télémétrie ouverte

Découvrez comment Open Telemetry normalise l’observabilité sur les journaux, les traces et les mesures sans vous verrouiller dans un seul outil de surveillance. Cette vidéo explique pourquoi l’adoption d’une norme ouverte à l’échelle du secteur permet une corrélation transparente du signal. Découvrez comment Open Telemetry prend en charge les piles existantes et fournit des informations opérationnelles de bout en bout pour une stratégie de surveillance unifiée.

* Standardisez l’observabilité avec OpenTelemetry sur tous les signaux
* Corréler les logs, les traces et les mesures pour accélérer l’analyse des causes premières
* Maintenir les outils de surveillance existants tout en unifiant les flux de données
