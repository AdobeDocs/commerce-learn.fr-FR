# Installation et configuration

Découvrez comment obtenir une observabilité de bout en bout dans Adobe Commerce et les applications personnalisées à l’aide d’une télémétrie normalisée et corrélée. Cette vidéo explique pourquoi la corrélation et la normalisation sont importantes. Elle présente le module Observability de Commerce et la bibliothèque de télémétrie en direct d’IO. Découvrez comment ces outils permettent une surveillance, une journalisation et un suivi unifiés pour un flux opérationnel transparent.

## À qui s&#39;adresse cette vidéo ?

* Développeur Adobe Commerce
* Architecte de solutions
* Responsable produit technique

## Contenu vidéo

* Normaliser et corréler la télémétrie pour une observabilité totale
* Utiliser le module d’observabilité de Commerce pour les signaux automatisés
* Extension de la surveillance avec la bibliothèque IO Live Telemetry pour les applications
