# Utilisation de l’outil de compatibilité de mise à niveau dans PhpStorm

{{only-for-on-prem-commerce-cloud}}

L’outil de compatibilité de mise à niveau (UCT) analyse les incompatibilités entre votre version actuelle et une version mise à niveau cible en quelques minutes. Le plug-in PhpStorm facilite l’exécution de l’outil à partir de votre IDE.

Dans cette vidéo :

* Fonctionnement de l’outil de compatibilité de mise à niveau
* Comment l’utiliser avec le plug-in Magento PhpStorm
* Comment lire les résultats HTML par gravité de module et nombre d’incompatibilités

## Contenu vidéo
