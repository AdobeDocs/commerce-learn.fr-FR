# Démonstration du kit de démarrage pour les intégrations back-office

Découvrez le kit de démarrage de l’intégration back-office d’Adobe Commerce. Ce tutoriel vidéo présente l’ensemble du processus de cet exemple d’intégration.

## Audience

* Toute personne ayant besoin de comprendre ce concept.
* les applications Commerce qui nécessitent l’intégration de leur application Adobe Commerce à un système back-end.

## Contenu vidéo

* Configuration d’un projet dans Adobe Developer Console
* Différents espaces de travail
* Ajout d’une API
* Types d’authentification
* Service Adobe I/O Runtime
* Configuration des événements I/O
* Connexion à un tiers pour synchroniser les données
