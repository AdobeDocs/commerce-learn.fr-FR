# Commerce Intelligence - Ensembles de filtres

Voulez-vous analyser uniquement certains éléments des données de votre entreprise ? Fatigué de faire un nettoyage manuel ? Dans Commerce Intelligence, vous pouvez utiliser la fonction de jeux de filtres pour simplifier ce processus.

Dans cette vidéo :

* Création d’un jeu de filtres
* Utilisation d’opérateurs de filtre
* Application d’un jeu de filtres à une mesure
* Application d’un jeu de filtres à une dimension de données calculées

>[!TIP]
>
>Si vous découvrez Commerce Intelligence, le [Guide de l’utilisateur de Adobe Commerce Intelligence](https://experienceleague.adobe.com/docs/commerce-business-intelligence/mbi/guide-overview.html){target="_blank"} fournit une vue d’ensemble complète pour obtenir de meilleures informations et résultats commerciaux grâce à l’agrégation, l’analyse et la visualisation des données.
