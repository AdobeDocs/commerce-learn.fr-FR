# Créer des tarifs de livraison personnalisés à l’aide d’outils d’IA - Créer la fonctionnalité

Découvrez comment utiliser les outils optimisés par l’IA d’Adobe pour rationaliser le workflow de développement et étendre les fonctionnalités de passage en caisse. Ce tutoriel porte sur l’intégration des frais d’expédition personnalisés à partir de systèmes externes à l’aide d’une API simulée sécurisée par une clé API. Découvrez comment configurer les URL de service, gérer la sécurité et activer les fonctionnalités de passage en caisse avancées. Ce guide complet explique comment l’IA accélère le développement logiciel, de la collecte des exigences à la planification de l’implémentation. Regardez la vidéo intégrée pour une présentation détaillée.

* Un agent d’IA analyse le contexte et utilise des compétences basées sur les rôles pour guider le développement.
* Le kit de démarrage fournit une structure et des modèles d’échantillon pour une création d’extension plus rapide.
* Les outils MCP exposent les commandes et les données utilisées par l’agent pour planifier et générer du code.
* L’IA automatise les exigences, l’architecture et les décisions d’implémentation.
