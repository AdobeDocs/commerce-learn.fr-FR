# Modification de l’URL d’une boutique

Chaque site web d’une installation Adobe Commerce ou Magento Open Source comporte une URL de base attribuée au storefront et une autre URL attribuée à l’administrateur. Vous pouvez suivre quelques étapes simples pour modifier l’URL de base de votre boutique.

## À qui s&#39;adresse cette vidéo ?

* Administrateurs de magasin
* Gestionnaires de sites Web
